# iJanda Test Cases

## TC-001 - Add place to favorites

**Preconditions:**
- User is logged in

**Steps:**
1. Open the app
2. Select a place
3. Click on "Add to favorites"

**Expected result:**
- The place is added successfully to the favorites list

## TC-002 - Search for a place

**Preconditions:**
- App is opened

**Steps:**
1. Go to search bar
2. Type a place name
3. Press search

**Expected result:**
- Results related to the search term are displayed

- ## TC-003 - View place details

**Preconditions:**
- At least one place is visible in the list

**Steps:**
1. Select a place from the list
2. Click on it

**Expected result:**
- Detailed information about the place is shown (description, location, etc.)

- ## TC-004 - Map loading

**Preconditions:**
- Internet connection available

**Steps:**
1. Open map section of the app

**Expected result:**
- Map loads correctly with markers visible
