# iJanda Bug Reports

## BUG-001 - Favorites button feedback unclear

**Severity:** Low

**Steps to reproduce:**
1. Open the app
2. Select a place
3. Click "Add to favorites"

**Actual result:**
- No clear confirmation message is shown

**Expected result:**
- User should see a confirmation like "Added to favorites"

**Notes:**
- This affects user experience but not functionality

## BUG-002 - Missing feedback after adding to favorites

**Severity:** Medium  
**Priority:** High  
**Type:** UI / Usability  

**Steps to reproduce:**
1. Open iJanda app
2. Select a place
3. Click "Add to favorites"

**Actual result:**
- No confirmation message or visual feedback is shown

**Expected result:**
- User should receive a confirmation message (e.g., "Added to favorites") or visual indicator

**Impact:**
- Users may not be sure if the action was successful
