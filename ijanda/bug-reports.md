# iJanda Bug Reports

## BUG-001 - Favorites button feedback unclear

**Severity:** Low

**Steps to reproduce:**
1. Open the app
2. Select a place
3. Click "Add to favorites"

**Actual result:**
- No clear confirmation message is shown

**Expected result:**
- User should see a confirmation like "Added to favorites"

**Notes:**
- This affects user experience but not functionality

## BUG-002 - No loading indicator on search

**Severity:** Medium

**Steps to reproduce:**
1. Open app
2. Perform a search

**Actual result:**
- No loading indicator appears while results are loading

**Expected result:**
- A loading spinner should be shown
